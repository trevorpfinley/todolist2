## How to Deploy to GitHub Pages

### Step 1: Create a GitHub Account
If you don't have one, go to https://github.com and sign up for free.

### Step 2: Create a New Repository
1. Click the "+" icon in the top-right corner
2. Select "New repository"
3. Name it something like `my-todo-list` (or anything you want)
4. Make it **Public**
5. **Do NOT** initialize with README
6. Click "Create repository"

### Step 3: Upload Your Files
1. On the repository page, click "uploading an existing file"
2. Drag and drop these three files:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Scroll down and click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to your repository's Settings (tab at the top)
2. Scroll down to the "Pages" section in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait a few minutes for GitHub to deploy your site

### Step 5: Access Your Website
Your website will be available at:
```
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```

For example: `https://johnsmith.github.io/my-todo-list/`

## Important Notes About Notifications

**Browser notifications only work when:**
- The website is open in a browser tab
- You've granted notification permissions
- Your browser supports notifications (Chrome, Firefox, Edge, Safari)

**Limitations:**
- Tasks are saved locally per browser (won't sync between phone and laptop automatically)
- Notifications won't work when the browser is closed
- You need to keep the tab open for reminders to trigger

## How to Use

1. **Add a task**: Type in the input box and click "Add Task" or press Enter
2. **Set a reminder**: Pick a date/time before adding the task
3. **Complete a task**: Click the checkbox next to the task
4. **Delete a task**: Click the "Delete" button
5. **Filter tasks**: Use All/Active/Completed buttons to filter your view
6. **Clear completed**: Remove all completed tasks at once

## Tips

- Allow notifications when your browser asks for permission
- Keep the tab open for reminders to work
- Bookmark the page for quick access
- Use it on both your phone and laptop by visiting the same URL
